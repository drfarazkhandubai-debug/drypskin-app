# App Store Screenshots

Capture screenshots from a physical device or simulator and place them here.

## iOS (App Store Connect)

| Device | Size |
|--------|------|
| iPhone 15 Pro Max (6.7″) | 1290 × 2796 px — **required** |
| iPhone 8 Plus (5.5″)     | 1242 × 2208 px — **required** |
| iPad Pro 12.9″ (if tablet support added) | 2048 × 2732 px |

## Android (Google Play Console)

| Type | Size |
|------|------|
| Phone screenshot | 1080 × 1920 px minimum |
| Feature graphic  | 1024 × 500 px (see assets/splash/) |
