# Splash Screen Assets

Place the following files here before running EAS Build:

| File | Size | Notes |
|------|------|-------|
| `splash.png` | 1284 × 2778 px | Universal splash — centred logo on white/cream background |
| `feature-graphic.png` | 1024 × 500 px | Google Play feature graphic (shown at top of store listing) |

**Best practice:**
- Use a centred DrypSkin logo on a white (`#FFFFFF`) or linen (`#F5F0EB`) background
- Keep the logo within the central 800 × 800 px safe zone so it renders well across screen sizes
- Update `app.json` → `splash.image` to point to your final splash file
